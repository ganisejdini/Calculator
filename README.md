# Calculator
This FPGA-based digital calculator system is a complete Verilog implementation consisting of a central control unit, a hardware-level instruction encoder, and an automated verification environment. The core architecture integrates a 32-bit Arithmetic Logic Unit (ALU) with a 16-bit accumulator register that stores the result of each operation and drives the physical LED outputs of the board. The design allows for data processing from the board's switches, where 16-bit input values are sign-extended to 32-bit format before any computation occurs. User interaction is managed through physical push-buttons, including a master reset to clear the accumulator and a confirm button to commit the current ALU result to memory. Operation selection is handled by a specialized encoding module that translates combinations of three input buttons into 4-bit ALU operation codes using purely combinational gate-level logic. This circuit utilizes basic NOT, AND, OR, and XOR gates to decode user commands, ensuring the ALU performs the specific arithmetic or logical task required in each clock cycle. By decoupling the control logic from the main processing path, the system allows for modular updates to the instruction set without modifying the central calculator core. To ensure design reliability, the system includes a comprehensive testbench that simulates real-world usage scenarios. The verification process includes clock generation, global reset procedures, and a sequence of automated test steps covering operations like addition, multiplication, and various logic gates. Each test step automatically validates the accumulator's state against expected values and reports detailed pass or fail messages to the simulation console.
